Advice Slip API

The Advice Slip JSON API is a public API that allows you to query a random piece advice

Love Calculator API

The love calculator takes 2 names and calculates the love compatability! Find out if you two are meant for each other!

Meme Generator

This API generates memes by choosing a preloaded image, and short top and bottom text sentences. Memes are served in the JPEG image file format.

This APIs used are the Advice Slip and the ImgFlip APIs. The Love Calculator API was not used as it no longer free.
